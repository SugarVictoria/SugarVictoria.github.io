# Sugar & Victoria Website

A responsive website built for Sugar & Victoria, an independent cafe in Stourport-on-Severn, Worcestershire.

## Overview

Sugar & Victoria is a modern cafe website designed to showcase the business’s menu, story, reviews, and visitor information in a clean and easy-to-navigate format. The site supports the cafe’s branding with a warm, welcoming style that reflects the in-person experience.

## Features

- Responsive design for desktop and mobile devices.
- Menu page with food and drink information.
- About page highlighting the cafe’s story and identity.
- Reviews page to display customer feedback.
- Visit page with location and contact details.
- Privacy and cookie policy pages.
- Hosted with a low-cost static deployment approach.
- Content managed through Netlify as a headless CMS.

## Tech Stack

- HTML
- CSS
- JavaScript
- JSON
- GitHub Pages
- Netlify

## What I Built

This project was developed to help a local independent business establish a professional online presence. The focus was on clear presentation, simple navigation, and a lightweight site structure that is easy to maintain and update.

## Impact

The website helped create a more polished digital presence for the cafe and provided a central place for customers to view the menu, learn about the business, and find practical visit information.

## Requirements

- A modern web browser.
- Internet access.
